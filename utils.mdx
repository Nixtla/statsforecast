Generate Synthetic Panel Series. 

Generates `n_series` of frequency `freq` of different lengths in the interval [`min_length`, `max_length`]. If `n_static_features > 0`, then each series gets static features with random values. If `equal_ends == True` then all series end at the same date. 



**Args:**
 
 - <b>`n_series`</b> (int):  Number of series for synthetic panel. 
 - <b>`freq`</b> (str, optional):  Frequency of the data, 'D' or 'M'. Defaults to 'D'. 
 - <b>`min_length`</b> (int, optional):  Minimum length of synthetic panel's series. Defaults to 50. 
 - <b>`max_length`</b> (int, optional):  Maximum length of synthetic panel's series. Defaults to 500. 
 - <b>`n_static_features`</b> (int, optional):  Number of static exogenous variables for synthetic panel's series. Defaults to 0. 
 - <b>`equal_ends`</b> (bool, optional):  Series should end in the same date stamp `ds`. Defaults to False. 
 - <b>`engine`</b> (str, optional):  Output Dataframe type ('pandas' or 'polars'). Defaults to 'pandas'. 
 - <b>`seed`</b> (int, optional):  Random seed used for generating the data. Defaults to 0. 



**Returns:**
 
 - <b>`DataFrame`</b>:  Synthetic panel with columns [`unique_id`, `ds`, `y`] and exogenous.
